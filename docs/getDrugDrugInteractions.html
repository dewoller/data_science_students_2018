Selection of interacting drugs from BNF file.

To begin with, the BNF file which was in pdf format, in which the names of the Drugs where coated in green colour, we first filtered and replaced those drugs name (i.e, Panadole -> ####Panadole%%%%) and converted the pdf file into text file. After conversion, we wrote a regular expression script in Perl to only extract the drugs name. Each drugs were having interactions with some other multiple drugs. For example, Abacavir -> RIBAVIRIN ; Abacavir -> TIPRANAVIR; Abacavir -> ORLISTAT. Also Tipranavir -> CLARITHROMYCIN ; Tipranavir -> RIFABUTIN ;Tipranavir -> RIFAMPICIN. The detail structure of pdf file to text file and further in our analysis is shown below:

read.table( 'data/drug_interactions.txt', sep='\t',header = FALSE) %>%
  as.tibble() %>% 
  mutate(V1=str_trim(str_to_upper(V1))) %>%
  mutate(V2=str_trim(str_to_upper(V2))) %>%
{ . } -> drug_drug_concordance

Binding drug-drug interactions.

We need to bind these 2 rows, so that these interaction can be removed for further analysis. The code to do this is just below:

drug_drug_concordance %>%

  bind_rows(drug_drug_concordance %>% select(drug2)) %>%

  {.} -> final_drugs
